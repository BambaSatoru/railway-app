# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog (https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2025-05-14

### Added
- Initial release of Railway App
- Express.js backend with basic routing
- PostgreSQL database integration
- GitHub Actions CI/CD workflow for Railway deployment
- GET `/` endpoint returning "Hello from Railway!"
- GET `/db` endpoint for database connectivity testing
- Procfile for Railway platform configuration
- MIT License
- Contributing guidelines
- Code of Conduct
- Security policy
- This CHANGELOG

### Features
- Automatic deployment to Railway on push to main branch
- Database connection pooling with pg library
- Proper error handling for database queries
- Environment variable configuration support
